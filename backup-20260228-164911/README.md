# 🎮 接住星星 - H5小游戏

一个使用HTML5 Canvas开发的浏览器小游戏，控制篮子接住掉落的星星和宝石，避开炸弹！

## 🎯 游戏特点
- **三种掉落物品**：星星(+10分)、宝石(+25分)、炸弹(-1生命)
- **渐进难度**：得分越高，掉落速度越快
- **多平台控制**：键盘 + 触摸屏支持
- **精美视觉效果**：渐变背景、粒子效果、得分动画
- **完整游戏功能**：开始、暂停、重新开始、游戏结束

## 🕹️ 游戏控制
### 键盘控制
- **← → 箭头键**：移动篮子
- **空格键**：暂停/继续游戏
- **Enter键**：开始游戏

### 触摸屏控制
- **触摸屏幕左右两侧**：移动篮子

## 🎨 游戏规则
1. 使用篮子接住掉落的星星和宝石获得分数
2. 避开红色的炸弹，否则会减少生命值
3. 每得100分升一级，掉落速度加快
4. 生命值为0时游戏结束
5. 尽可能获得高分！

## 🔧 本地运行
