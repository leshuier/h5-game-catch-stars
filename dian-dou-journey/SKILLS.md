## ✅ 已掌握技能

### 1. 摄像头控制 📸

**功能：** 从摄像头拍摄照片

**实现方式：**
```bash
ffmpeg -f v4l2 -i /dev/video0 -frames 1 output.jpg
```

**使用场景：**
- 安全检查
- 记录环境
- 视觉辅助

**文件位置：** `/tmp/camera_snapshot.jpg`

### 2. 终端游戏开发 🎮

**功能：** 创建可玩的俄罗斯方块游戏

**实现方式：**
- Python + 终端 UI
- 支持键盘控制
- 计分系统

**文件位置：**
- `tetris.py` - GUI 版本（需要 pygame）
- `tetris_terminal.py` - 终端版本（已测试可用）

**控制方式：**
- ← → 移动
- ↑ 旋转
- ↓ 快速下落
- Space 硬降落
- R 重新开始
- Q 退出
